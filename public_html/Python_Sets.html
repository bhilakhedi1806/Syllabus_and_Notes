Python Sets

Python Sets

  • A set is a collection which is unordered and unindexed.
  • In Python, sets are written with curly brackets.
  • Example
  • Create a Set:
            thisset = {"apple", "banana","cherry"}
            print(thisset)
  • Output
    •             ['apple','banana','cherry']

    Accessing Item

    • Set items cannot access by an index or a key.
    • But can access through a for loop, or ask if a specified value is present in a set, by using the in keyword.
  • Example
  • Loop through the set, and print the values:
    •                    thisset = {"apple", "banana", "cherry"}
                   for x in thisset:
                   print(x)
  • Output
    •                     apple
                    banana
                    cherry

    Add Items

    • To add one item to a set use the add() method.
    • To add more than one item to a set use the update() method.
  • Example
  • Add an item to a set, using the add() method:
    •                   thisset = {"apple", "banana", "cherry"}
                  thisset.add("orange")
                  print(thisset)
  • Output
    •                     {'apple', orange', 'banana', 'cherry'}
  • Example
  • Add multiple items to a set, using the update() method:
    •                
                    thisset = {"apple", "banana", "cherry"}
                    thisset.update(["orange", "mango", "grapes"])
                    print(thisset)
  • Output
    •                     {'banana', 'apple', 'cherry', 'mango', 'grapes', 'orange'}

    Length Of A Set

    • To determine how many items a set has, use the len() method:
  • Example
  • Get the number of items in a set:
    •                 thisset = ("apple", "banana", "cherry")
                print(len(thisset))
  • Output
    •                 3

    Remove Item

    • To remove an item in a set, use the remove(), or the discard() method.
  • Example
  • Remove "banana" by using the remove() method
    •                 thisset = ("apple", "banana", "cherry")
                thisset.remove("banana")
                print(thisset)
  • Output
    •                {'apple', 'cherry'}
  • Note: If the item to remove does not exist, remove() will raise an error.
    • To remove an item in a set, use the remove(), or the discard() method.
  • Example
  • Remove "banana" by using the discard() method:
    •                 thisset = ("apple", "banana", "cherry")
                thisset.discard("banana")
                print(thisset)
  • Output
    •                 {'apple', 'cherry'}
  • Note: If the item to remove does not exist, discard() will NOT raise an error.
    • You can also use the pop(), method to remove an item, but this method will remove the lastitem.
    • Remember that sets are unordered, so you will not know what item that gets removed.
    • The return value of the pop()method is the removed item.
  • Example
  • Remove the last item by using the pop()method:
    •                 thisset = {"apple", "banana", "cherry"}
                x = thisset.pop()
                print(x)
                print(thisset)
  • Output
    •                 banana
                {'apple', 'cherry'}

    Join Two Sets

    • There are several ways to join two or more sets in Python.
    • You can use the union()method that returns a new set containing all items from both sets, or the update()method that inserts all the items from one set into another:
  • Example
  • The union()method returns a new set with all items from both sets:
    •                 set1 = {"a", "b" , "c"}
                set2 = {1, 2, 3}
                set3 = set1.union(set2)
                print(set3)
  • Output
    •                 {'c', 2, 'b', 1, 'a', 3}
  • Example
  • The update()method inserts the items in set2 into set1:
    •                 set1 = {"a", "b" , "c"}
                set2 = {1, 2, 3}
                set1.update(set2)
                print(set1)
  • Output
    •                 {'c', 2, 'b', 1, 'a', 3}
  • Note Both union() and update() will exclude any duplicate items.

    • Python Set Operations

    • Following set Operations:
  • union, intersection, difference and symmetric difference
    • In Python, below quick operands can be used for different operations.
  • | for union.
    & for intersection.
    – for difference
    ^ for symmetric difference
  • Example
    •                 A = {0, 2, 4, 6, 8}
                B = {1, 2, 3, 4, 5}
  • Example
    •                 # union
                print("Union :", A | B)
  • Output
    •                 Union : {0, 1, 2, 3, 4, 5, 6, 8}
  • Example
    •                 # intersection
                print("Intersection :", A & B)
  • Output
    •                 Intersection :{2, 4}
  • Example
    •                 # difference
                print("Difference :", A - B)
  • Output
    •                 Difference : {8, 0, 6}
  • Example
    •                 # symmetric difference
                print("Symmetric difference :", A ^ B)
  • Output
    •                 Symmetric difference : {0, 1, 3, 5, 6, 8}
