Python Variable Scope
  • In Python Code, these are various unique namespaces defined, we may not be able to access all of them from every part of the program.
  • The concept of scope comes into play.
  • A scope is the portion of a program from where a namespace can be accessed directly without any prefix.
  • At any given moment, there are at least three Variable scopes.
  • local namespace
  • global namespace
  • built-in namespace
  • When a reference is made inside a function, the name is searched in the local namespace, then in the global namespace and finally in the built-in namespace.
  • If there is a function inside another function, a new scope is nested inside the local scope.
  • Example of Scope and Namespace in Python

def outer_function():

      b = 20  

   print(“b:=“,b)

a = 10

outer_function()

print(“a:=“,a)

  • OUTPUT:

b:=20

a:=10

  • Here, the variable ais in the global namespace. Variable bis in the local namespace of outer_function()

def outer_function():

    global a

    a = 20   

    def inner_function():

   global a

      a = 30

        print('a =', a)

    inner_function()   

    print('a =', a)

a = 10

outer_function()

print('a =', a)

  • OUTPUT:

a:=30

a:=30

Here, all references and assignments are to the global a due to the use of keyword global
