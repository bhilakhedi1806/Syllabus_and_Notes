Module
Concept of modules in python

  • A module is a file consisting of Python code.
  • A module can define functions, classes and variables.
  • A module can also include runnable code.
  • Consider a module to be the same as a code library.
  • A file containing a set of functions you want to include in your application.

Create a Module

  • To create a module just save the code you want in a file with the file extension
    •  .py:
  • Example
    • Save this code in a file named mymodule.py
      •    defmessage(name):
         print(“Welcome to, "+ name)

Use Modules

  • Now we can use the module we just created, by using the importstatement:
  • Example
  • Import the module named mymodule, and call the greeting function:
    •  importmymodule
     
     mymodule.message(“Python")
  • Note: When call a function from a module, use the syntax: module_file_name.function_name.

Re-naming a Module

  • You can create an alias when you import a module, by using the askeyword:
  • Example
  • Create an alias for mymodulecalled mx:
import mymodule as mx
 
 
 mymodule.message(“Python")
