Variable Names
  • Variable Names
  • A variable can have a simple name (like a and b) or a more descriptive name (student, car, number, etc)
  • A variable name must start with a letter or the underscore character
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case-sensitive (age, Age and AGE are three different variables)
  • Example :-
              #Valid variable names:
              var_name = "Python"
              varname = "Python"
              _var_name = "Python"
              _varname = "Python"
              _2varname = "Python"
              varname2 = "Python"
            
              #Invalid variable names:
              2varname = "Python"
              var-name = "Python"
              var name = "Python"
  • Python allows to assign values to multiple variables in one line:
  • Example :-
              x , y , z = "Red" , "Green" , "Blue"
              print(x)
              print(y)
              print(z)
  • And also allow to assign same value to multiple variable in one line
              x=y=z = "Red"
              print(x)
              print(y)
              print(z)
